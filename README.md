# 📚 Library Management System

A simple Python-based Library Management System developed using file handling and CSV storage. This project helps manage books and student records through a menu-driven program.

## ✨ Features

* Add Books
* Display Books
* Issue Books
* Return Books
* Search Books
* Generate Reports
* CSV File Handling
* Exception Handling

## 🛠 Technologies Used

* Python
* CSV Module
* File Handling
* Functions
* Loops
* Lists & Dictionaries

## 📂 Files Included

* `library_management.py`
* `books.csv`
* `issued_books.csv`

## ▶ How to Run

1. Open the project in VS Code or any Python IDE.
2. Run `library_management.py`
3. Use the menu options to manage library records.

## 📖 Concepts Covered

* Menu Driven Programs
* File Handling
* Data Storage
* Exception Handling
* Python Functions
